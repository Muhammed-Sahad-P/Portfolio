MyProject

FootFusion

E-Commerce Application

Project Overview: The E-commerce application is a full-stack solution designed to provide a seamless online shopping experience. This project focuses on user-friendly interfaces and secure transactions, leveraging modern technologies for optimal performance.

Key Features:

  • User Authentication: Implemented secure user login and registration with JWT-based token authentication. Protected routes ensure that sensitive data is accessible only to authenticated users.
  • Product Management: Efficiently fetched product data from MongoDB and displayed it in responsive React components styled with Tailwind CSS. The application includes cart and wishlist functionalities to enhance user engagement.
  • Cart & Wishlist: Managed real-time cart and wishlist operations using React hooks (useState, useEffect) and RESTful APIs, with global state management facilitated by the Context API for a smooth user experience.
  • Order & Payment: Integrated Razorpay for secure online payments, allowing users to create orders, process payments, and track their order history with ease.
  • Backend Architecture: Developed RESTful APIs in Node.js and Express, handling user, product, cart, wishlist, and payment functionalities, ensuring a robust backend to support the application.
  • Database Management: Utilized MongoDB to effectively manage user accounts, products, orders, and transactions, ensuring data integrity and accessibility.

Tech Stack:

  • Frontend: React, Tailwind CSS, Razorpay
  • Backend: Node.js, Express.js, MongoDB, JWT
  • Tools: Git, GitHub, Postman, Razorpay API
  • Deployment: Deployed on Render, with careful management of environment variables for API keys and secure operations.
